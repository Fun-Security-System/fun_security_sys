<a href="#Raspi-Project-16">Installation</a>

1. Download

$ git clone https://github.com/Raspi-Project-16/fun_security_sys-opencv-ws2812.git

2. Compile

$ cd raspi_project_16
$ cmake .
$ make

3. Run

3.1 Save your photos into the dataset folder

3.2 Train the dataset

$ sudo ./train

3.3 Run the system

$ sudo ./run
